⚡ LANGKAH CEPAT:
----------------
1. Buka Replit.com → Login
2. Create Repl → Node.js
3. Upload semua file di atas
4. Klik ikon kunci (Secrets) → Tambahkan 3 secrets
5. Klik tombol "Run"
6. Test di Telegram dengan /start
7. Kirim link Instagram/TikTok/Twitter/YouTube/Pinterest
8. Bot akan kirim FILE video/foto (bukan link)

🌐 KEEP BOT ONLINE (AGAR TIDAK SLEEP):
--------------------------------------
1. Buka UptimeRobot.com → Buat akun gratis
2. Add New Monitor:
   - Type: HTTP(s)
   - URL: (copy dari Webview Replit)
   - Interval: 5 minutes
3. Save
